## Problem

PlantPriceTracker has no About page. This is the #1 trust signal Google
looks for on affiliate sites (E-E-A-T). Without it, affiliate program
applications will be weaker and search rankings suffer.

## Scope Decisions

- Static content page (same pattern as disclosure.html / privacy.html)
- Placeholder text for personal/company details — Brandon fills in later
- Sections: Who runs this, What we do, How prices are collected, Editorial standards, Contact
- E-E-A-T signals: real person/company context, methodology transparency, update frequency
- Nav link added to all pages via base.html
- Footer link added alongside Disclosure and Privacy

## Out of Scope

- Team member photos or headshots
- Dynamic "last updated" date from build pipeline
- Schema.org Person/Organization markup (can be added later)

---

## Technical Approach

Follow the exact pattern used by disclosure.html and privacy.html:
1. Jinja2 template extending base.html with title/description/content blocks
2. Render call in build.py passing canonical_url
3. Static URL added to sitemap_urls list
4. Nav link in base.html header, footer link in footer-links div

## Files Likely Touched

- `templates/about.html` (new)
- `templates/base.html` (nav + footer link)
- `build.py` (render call + sitemap entry)
- `tests/test_build_pages.py` (new test class)
